Commandes Pratiques
Nom Fonction Syntaxe
Team Permet de créer deux équipes équilibrées à partir des noms des joueurs. Cependant, le nombre de joueur doit être pair. *team "joueur1,joueur2,joueur3,joueur4,..."
Poll Permet de générer un sondage fermé ou ouvert *poll "question fermé"
*poll "question ouverte" "choix1,choix2,choix3 ..."
Random Renvoie un nombre aléatoire entre deux borne définit par l'utilisateur *random nombre1 nombre2
Command Envoie un lien vers cette page *command
Site Envoi un lien vers notre site *site
Help Affiche toutes les commandes disponnibles *help
Commandes De Modérations
Nom Fonction Syntaxe
Clear Permet d'effacer un nombre precis de messages dans le salon textuel *clear [nombres de messages a supprimer]
Kick Permet de kick une personne en ajoutant une raison *kick @membre#000 raison du kick optionel
Ban Permet de ban une personne en ajoutant une raison *ban @membre#000 raison du ban optionel
Commandes Fun
Nom Fonction Syntaxe
Pile ou Face Lance un jeu de pile ou face contre le bot *pileface votre choix(pile ou face)
Juste prix Lance le célèbre jeux du juste prix *justeprix
Gif Le bot envoie un gif aléatoire dans le salon textuel *gif
